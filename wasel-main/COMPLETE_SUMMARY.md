## 🚀 خطة التنفيذ الفورية

### المرحلة 1: إعداد Live PayPal (⏱️ 30 دقيقة)

```bash
# الخطوة 1: الحصول على Credentials
📌 اذهب: https://developer.paypal.com/dashboard/
📌 انسخ: Client ID و Secret (Live)

# الخطوة 2: إضافة إلى Supabase
📌 اذهب: https://app.supabase.com/
📌 Settings → Functions → Environment Variables
📌 أضف PAYPAL_CLIENT_ID و PAYPAL_CLIENT_SECRET و PAYPAL_ENV=live

# الخطوة 3: النشر
npm run deploy:functions
```

### المرحلة 2: الاختبار (⏱️ 15 دقيقة)

```bash
# اختبر على Sandbox أولاً
# قيّم PAYPAL_ENV = sandbox في Supabase

npm run dev
# جاهز للاختبار على http://localhost:5173
```

### المرحلة 3: التفعيل (⏱️ 5 دقائق)

```bash
# بعد نجاح Sandbox:
# غيّر: PAYPAL_ENV = live في Supabase
# الآن بدّء الدفع الفعلي!
```

### المرحلة 4: Android (⏱️ 20 دقيقة)

```bash
npm run build:apk
# ثبّت على جهاز Android
# جرّب الدفع من التطبيق
```

## 🆘 دعم ودعم السريع

### المشاكل الشائعة وحلولها

| المشكلة | السبب | الحل |
|--------|------|------|
| "Unauthorized" | Credentials خاطئة | تحقق من Supabase env vars |
| "INVALID_PARAMETER_SYNTAX" | Origin = null | استخدم Fallback domain |
| Gradle won't sync | SDK قديم | حمّل SDK 34+ |
| APK not installing | Permissions | تحقق من AndroidManifest |
| PayPal not redirecting | HTTPS لم يفعّل | استخدم HTTPS فقط |
| الدفع لا يكتمل | Functions لم تنشر | اشغل deploy:functions |
