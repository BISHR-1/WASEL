# 🧪 دليل الاختبار السريع - الدفع الحقيقي برقم صغير

## ✅ تم التحضير!

تم تعيين **رسم التوصيل = 0** مؤقتاً لاختبار آمن

---

## 🚀 الاختبار الآن (5 دقائق)

### الخطوة 1️⃣: شغّل التطبيق

```bash
npm run dev
```

### الخطوة 2️⃣: أضف منتج

```
1. اضغط على أي منتج
2. اختر الكمية (1)
3. أضفه للسلة
```

### الخطوة 3️⃣: اذهب للسلة

```
1. اضغط على أيقونة السلة
2. تحقق: رسم التوصيل = 0 ✅
3. مثال: منتج $1 = $1 فقط (بدون رسم)
```

### الخطوة 4️⃣: ادفع

```
1. اضغط "Pay with PayPal"
2. سيفتح PayPal
3. اختر: بطاقة ائتمان/خصم
```

### الخطوة 5️⃣: استخدم بطاقة اختبار

```
منطقة الرمز البريدي:
🇺🇸 أمريكا (مثلاً)

بطاقة اختبار:
4111 1111 1111 1111

الصلاحية:
أي تاريخ مستقبلي (مثل 12/25)

CVV:
أي 3 أرقام (مثل 123)

الاسم:
أي اسم (مثل Test User)
```

### الخطوة 6️⃣: اكمل

```
1. اضغط "Pay"
2. انتظر المعالجة
3. ✅ يجب أن تذهب لصفحة "Payment Success"
```

## ⚠️ في حالة مشكلة

### المشكلة: "Payment Failed"

**الحل:**
```
1. تحقق من البيانات:
   - رقم البطاقة: 4111111111111111
   - التاريخ: مستقبلي
   - CVV: أي 3 أرقام

2. جرب مرة ثانية

3. إذا لم تنجح:
   - تأكد من: PAYPAL_ENV = sandbox
   - تحقق من: الـ credentials صحيحة
```

### المشكلة: "Cancel لم يعود للسلة"

**الحل:**
```
1. تأكد من الإصلاح السابق ✅
2. قد تحتاج لتحديث الـ Functions
   npm run deploy:functions
```
