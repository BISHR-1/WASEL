# مشروع تطبيق درعا - الهيكل البرمجي والميزات الذكية

## نظرة عامة
تطبيق React لتوصيل الطلبات من درعا، مع ميزات ذكية متقدمة وتكامل مع n8n للأتمتة.

## الهيكل البرمجي
```
src/
  pages/
    Home.jsx - الصفحة الرئيسية مع Carousel وتوصيات
    Categories.jsx - الفئات مع Grid
    Product.jsx - صفحة المنتج مع تفاصيل وإضافة للسلة
    Cart.jsx - السلة مع ميزة "الطلب لشخص آخر"
    Profile.jsx - Dashboard مع الطلبات والولاء
  components/
    Carousel.jsx - بانرات متحركة
    ProductCard.jsx - بطاقة المنتج مع صورة وسعر
    DiscountBadge.jsx - شارة الخصم
    NeighborhoodSection.jsx - أقسام حسب الأحياء
    Mascot.jsx - الشخصية غير البشرية
    CartItem.jsx - عنصر في السلة
    ProgressBar.jsx - شريط التقدم لتتبع الطلب
    Loyalty.jsx - نظام الولاء
    CategoryGrid.jsx - شبكة الفئات
  api/
    cartApi.js - APIs للسلة
    productApi.js - APIs للمنتجات
    userApi.js - APIs للمستخدم
    recommendationApi.js - APIs للتوصيات
    n8nService.js - خدمة التكامل مع n8n
  hooks/
    useCart.jsx - إدارة السلة
    useUser.jsx - إدارة المستخدم
    useRecommendation.jsx - التوصيات الذكية
  utils/
    helpers.js - مساعدات عامة
    formatters.js - تنسيق البيانات
  services/
    notificationService.js - إرسال الإشعارات
    loyaltyService.js - منطق الولاء
    trackingService.js - تتبع الطلبات
```

## الميزات الرئيسية
1. **الصفحة الرئيسية**: Carousel، أقسام مريحة للعين، عرض الأصناف مع خصم 50%، إبراز "درعا"، Mascot، توصيات ذكية.
2. **الفئات**: Grid واضح، صور حقيقية، انتقال للمنتجات.
3. **صفحة المنتج**: صورة واضحة، تفاصيل، أزرار السلة مع تأثيرات.
4. **السلة**: تعديل الكميات، حذف، مجموع، إكمال الطلب أو لشخص آخر.
5. **الملف الشخصي**: Dashboard مع بيانات، طلبات حالية/سابقة، نقاط الولاء، تتبع، تسجيل خروج.

## الميزة المحورية: "أهل درعا يجهّزون الطلب — المغترب يدفع"
- شخص في درعا يختار أصناف ويرسل رابط للمغترب.
- المغترب يدفع ويتم التنفيذ محليًا.
- رسالة WhatsApp عاطفية: "هذه الأصناف اختارها أهلك في درعا 🤍 بمسة منك… الفرح يوصل لبابهم."

## الميزات الذكية (جاهزة لـ n8n)
- **محرك توصيات**: حسب السلوك، الوقت، الحي.
- **عروض ديناميكية**: أصناف راكدة، مستخدم غير نشط، حسب المنطقة.
- **نظام ولاء**: نقاط، مكافآت، هدايا مخفية.
- **تتبع الطلب**: حالات واضحة، Progress bar، تحديثات فورية.
- **طلب جماعي**: دعوة أصدقاء.
- **الطلب لشخص آخر**: إرسال رابط.
- **إشعارات ذكية**: غير مزعجة، عبر n8n.
- **تقييم ذكي**: منع التقييمات الوهمية.

## التكامل مع n8n
- Webhooks لاستقبال الأحداث (إضافة للسلة، طلب جديد).
- REST APIs لإرسال البيانات إلى n8n.
- أتمتة الإشعارات، التوصيات، العروض.

## المتطلبات التقنية
- فصل UI، Logic، Services.
- كود نظيف وقابل للتوسعة.
- APIs واضحة لكل ميزة.
- جاهزية للإطلاق كمنتج رسمي.
