
## 🆘 مشاكل شائعة والحل

### ❌ "Gradle project needs to be built" لا تختفي

**الحل:**
```
1. File → Invalidate Caches and Restart
2. انتظر حتى يعيد فتح
3. ثم اضغط Sync Now مرة ثانية
```

### ❌ "gradle not found" أو أخطاء تحميل

**الحل:**
```
1. File → Settings (Ctrl+Alt+S)
2. Appearance & Behavior → System Settings → Android SDK
3. اضغط "Edit" إلى جانب SDK Path
4. تأكد المسار صحيح
5. ثم اضغط Sync Again
```

### ❌ "Emulator not found" (بدون جهاز حقيقي)

**الحل:**
```
1. أول مرة استخدم جهاز حقيقي (أسهل)
2. أو: Tools → Device Manager
3. اضغط: Create Device
4. اتبع الخطوات
5. لكن يحتاج RAM كثير
```

### ❌ "Sync failed" أو أخطاء أخرى

**الحل الكامل:**
```bash
# اذهب لمجلد android
cd android

# نظّف الملفات القديمة
./gradlew clean

# عودة لـ Android Studio وكرر Sync
```
