## 7. حالات الاستثناءات

```
الحالة الخاصة: StaffLogin

StaffLogin يمكن:
├─ أي شخص (لم يسجل دخول) يمكنه الدخول ✅
├─ موظف مسجل دخول يمكنه إعادة الدخول (اختياري)
│  └─ سيُفقد الجلسة الحالية
└─ لكن لا نظهر الرابط للموظفين في الملاح بشكل افتراضي
   └─ لأنهم بالفعل مسجلين دخول
```

## 8. دورة حياة الجلسة

```
1. عميل جديد (لا جلسة)
   │
   ├─ يحاول AdminPanel
   │  → ProtectedRoute: لا جلسة ❌
   │  → رسالة: "سجل دخول أولاً"
   │
   └─ يذهب لـ StaffLogin
      │
      └─ يختار دوره + يدخل بيانات
         │
         └─ loginAdminUser() ✅
            │
            └─ localStorage.set('admin_session')
               └─ جلسة نشطة الآن

2. موظف مسجل دخول (له جلسة)
   │
   ├─ يدخل /DriverPanel (دوره صحيح)
   │  → ProtectedRoute: لديه جلسة + دور صحيح ✅
   │  → اعرض الصفحة
   │
   └─ يدخل /AdminPanel (دوره خاطئ)
      → ProtectedRoute: لديه جلسة لكن دور خاطئ ❌
      → رسالة: "دور خاطئ"

3. الجلسة تنتهي (24 ساعة أو تسجيل خروج)
   │
   └─ localStorage.remove('admin_session')
      │
      └─ يحاول AdminPanel
         → ProtectedRoute: لا جلسة ❌
         → رسالة: "سجل دخول أولاً"
         → يعود لـ StaffLogin
```
